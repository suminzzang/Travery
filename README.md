## 주요 기능

주요 기능 :
- 카카오맵 api를 활용하여 실시간 여행지 탐색 가능
- Spring boot를 활용한 BackEnd 서버 구현
- Vue를 활용한 FrontEnd 구현
- JWT를 활용한 로그인 구현

## 개발 환경
### BackEnd
- Eclipse
- Spring Boot 2.7.12
- Java 8
- Mysql
- Mybatis

### FrontEnd
- Visual Studio Code
- Vue2
- nodejs 14
